## This Repo contains the code used for the presentation assignment in MB5370 
The presentation aims to visualize changes in hard coral cover of inshore reefs over the course of the AIMA long term monitoring program and to link these chances to annual dishcharge values from all rivers in given NRM regions
- Coral and Environmental Parameters Datasets: https://doi.org/10.25845/
- Coral Dataset with Coordinates for Sample Sites: https://doi.org/10.25845/5cc64f29b35a1
